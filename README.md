## 🚀 Deploy in 15 Minutes

```bash
git clone https://github.com/farmazmhs-creator/arusinovasicrm.git
cd arusinovasicrm
npm install
# Edit .env.local with Supabase + Resend keys
vercel --prod
```

**See [DEPLOYMENT.md](./DEPLOYMENT.md) for detailed instructions.**
